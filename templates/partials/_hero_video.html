{% comment %} Progressive hero background video. Any element with data-hero-video="/static/..." that contains an poster gets a muted/looping injected over that poster. Desktop only (>=861px); skipped for prefers-reduced-motion + data-saver (mobile/those users keep the static poster = the LCP element). Revealed on loadeddata/canplay (not just 'playing'). Some browsers (Brave, strict autoplay) block muted autoplay until interaction, so we also kick off playback on the first user gesture. ?vdebug=1 shows a live state readout. {% endcomment %}
