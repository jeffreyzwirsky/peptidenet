{% comment %} Progressive hero background video. Any element with data-hero-video="/static/..." that contains an poster gets a muted/looping injected over that poster. Runs on all viewports incl. mobile; skipped only for prefers-reduced-motion + data-saver (those users keep the static poster = LCP). Revealed on loadeddata/canplay (not just 'playing'). Some browsers (Brave, strict autoplay) block muted autoplay until interaction, so we also kick off playback on the first user gesture. ?vdebug=1 shows a live state readout. {% endcomment %}
