{% comment %} Progressive hero background video. Any element with data-hero-video="/static/..." that contains an poster gets a muted/looping injected over that poster. Desktop only (>=861px); skipped for prefers-reduced-motion + data-saver, so mobile/those users keep the static poster (the LCP element). The video is revealed as soon as a frame is available (loadeddata/canplay) — NOT only on 'playing' — to avoid an opacity:0 <-> deferred-autoplay deadlock. Append ?vdebug=1 to the URL to show a live state readout for troubleshooting. {% endcomment %}
